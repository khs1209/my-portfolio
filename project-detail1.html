팀 협업 프로젝트

기간: 2024.05.07 ~ 2024.06.25

설명: 본 프로젝트에서는 MySQL 서버와 연동하여 사용자 접근성을 높이고, YOLO 객체 검출을 활용한 실시간 데이터 확인 및 장애물 탐지 시스템을 구축하였습니다. 또한, ROS 기반 웹캠을 활용하여 화면 캡처 및 객체 검출을 수행하고, YOLO 학습 모델을 적용하여 객체 목록을 확인하였습니다.

사용 기술: Python, TensorFlow, OpenCV, YOLOv8, ROS, MySQL, PyQt5, HTML/CSS, JavaScript

  • 포트폴리오: github 링크

    • 주요 기능:

    • QR 코드 인식 및 OpenCV-YOLO 기반 장애물 탐지
    • AI 모델링 및 데이터 분석
    • 로그인 및 회원가입 시스템 구현 (MySQL 연동)
    • 사용자 대시보드 및 관리자 대시보드 분리
    • 웹캠을 활용한 AI 사물 인식 테스트
    • Arduino AI 자율주행 자동차 개발 및 장애물 감지
    • PyQt5를 활용한 GUI 컨트롤러 제작
    • ROS 기반 라인 트레이싱 자율주행 및 웹 인터페이스 수동 주행
