캘린더 프로젝트

기간: 2025.02.14 ~ 2025.02.21

설명: 사용자가 일정을 추가, 수정, 삭제할 수 있는 웹 기반 캘린더 애플리케이션을 개발했습니다.

사용 기술: React, Node.js, MongoDB

포트폴리오: github 링크

프로젝트 배경 및 추가 기능

초기에는 출시 예정 게임의 일정과 이벤트 관리를 위한 ‘게임 일정 캘린더’로 기획되었으나, 다양한 사용자들의 일정 관리 요구에 맞추어 게임에 국한되지 않고 전반적인 일정 관리에 초점을 맞추게 되었습니다.

  • 일정 등록, 수정, 삭제 기능
  • 월별, 주간, 일간 보기 지원
  • 검색 및 필터링 기능 강화
  • 사용자 친화적 인터페이스 및 반응형 디자인
  • 개인, 팀, 프로젝트 관리자를 위한 효율적 도구 제공

이 애플리케이션은 사용자가 손쉽게 일정을 관리하고, 중요한 이벤트나 회의를 놓치지 않도록 도와주는 것을 목표로 합니다.
